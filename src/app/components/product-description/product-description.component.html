ตัวช่วยการประมูล

• เมื่อคุณเพิ่มสินค้าชิ้นนี้ลงในรถเข็นด้วยวิธีการ Auto Win สินค้านี้จะยังไม่จบการประมูล

• เมื่อคุณชำระเงินสำหรับสินค้านี้เรียบร้อย คุณจะเป็นผู้ชนะการประมูล การประมูลจะจบลงและสินค้าชิ้นนี้จะเป็นของคุณ

• ระบบจะทำการประมูลสู้อัตโนมัติหากคุณไม่ใช่ผู้ชนะการประมูลสินค้าชิ้นนี้

• หากคุณไม่ทำการชำระเงินสำหรับสินค้านี้ที่เพิ่มลงในรถเข็นด้วยวิธีการ Auto Win และมีผู้ชนะการประมูลการประมูลจบลงก่อนที่คุณจะทำการชำระเงินสินค้าชิ้นนี้จะตกเป็นของผู้ชนะการประมูลทันที
